# Calculadora em C

Este é um projeto simples de uma calculadora em C que realiza as quatro operações básicas: adição, subtração, multiplicação e divisão.

## Como compilar e executar

Você pode compilar e executar o programa de duas formas:

### Usando GCC diretamente

```bash
gcc calculadora.c -o calculadora
./calculadora
```

### Usando o Makefile

O Makefile simplifica o processo.

**Para compilar:**
```bash
make
```

**Para executar:**
```bash
./calculadora
```

**Para limpar os arquivos gerados:**
```bash
make clean
```
